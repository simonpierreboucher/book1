3  Randam Variables

We will present the moments of a random variable, like the mean and the variance. We will also do an overview of the most important distributions. Finally, we will look at probability functions, such as the density function and the cumulative density function.

3.1 Expected value and the mean

We assume that a random variable \(Y\), can take \(k\) possible values.

\[ y_1, y_2, y_3, ......, y_{k-1},y_k \]

Where \(y_1\) represents the first possible value for \(Y\), \(y_2\) represents the second possible value for \(Y\), up to \(y_k\) representing the \(K^{th}\) possible value for \(Y\). Each of the possible values will be associated with a probability that it is the value taken by \(Y\). More precisely, the probability that \(Y = y_1\) will be presented by \(p_1\), the probability that \(Y = y_2\) will be presented by \(p_2\) and so on. By having all the possible values as well as its respective probability for our random variable Y, it is possible for us to find the expected value of \(Y\), represented by \(E(Y)\).

\[ \mu_Y=E(Y) = y_1 p_1 + y_2 p_2 + \cdots + y_k p_k = \sum_{i=1}^k y_i p_i \]

3.2 Variance and standard-deviation

Given that we have found the expected value of Y, we will be able to find its variance, represented by \(Var(Y)\).

\[ \text{Var}(Y) = E\left[(Y-\mu_Y)^2\right] = \sum_{i=1}^k (y_i - \mu_Y)^2 p_i \]

Then we can find the standard deviation of Y by taking the square root of our variance. \[ \sigma_Y=\sqrt{\sigma_Y^2}=\sqrt{Var(y)} \]

3.3 Continuous Random Variable

If the random variable Y is continuous, we can also find its expectation and its variance. A random variable will always be associated with a probability density function. In the case of our random variable Y, its probability density function will be represented by \(f_Y (y)\). The probability that
\(Y\) falls between \(a\) and \(b\) is found as follows. \[ P(a \leq Y \leq b) = \int_a^b f_Y(y) \mathrm{d}y \] An important property of the probability density function is that the set of possible probabilities must add up to 1.
\[ P(-\infty \leq Y \leq \infty) = 1 \]

\[ \int_{-\infty}^{\infty} f_Y(y) \mathrm{d}y = 1 \]

We can therefore define the expectation of our random variable \(Y\) as follows. \[ E(Y) = \mu_Y = \int y f_Y(y) \mathrm{d}y. \] As well as its variance

\[ \text{Var}(Y) = \sigma_Y^2 = \int (y - \mu_Y)^2 f_Y(y) \mathrm{d}y. \]

3.4 Normal distribution

The normal distribution is one of the best known and used given its simplicity. Indeed, it can be completely characterized by its mean \(\mu\) and its variance \(\sigma^2\). Usually it is represented as follows. \[ \mathcal{N}(\mu,\sigma^2) \]

The probability density function of a normal distribution is represented by the following equation.

\[ f(x) = \frac{1}{\sqrt{2 \pi} \sigma} \exp{-(x - \mu)^2/(2 \sigma^2)}. \]
